``print_sorted`` module
================================

Using ``MyList``
----------------

    >>> MyList = __import__("1-my_list").MyList

Now use it:

    >>> l = MyList()
    >>> l
    []

    >>> l.append(1)
    >>> l
    [1]

    >>> l.append(5)
    >>> l.append(1)
    >>> l
    [5, 1]

    >>> l.print_sorted()
