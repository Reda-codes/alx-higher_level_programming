``print_sorted`` module
================================

Using ``MyList``
----------------

import ``MyList`` from the ``1-my_list`` module:

    >>> MyList = __import__("1-my_list").MyList

Now use it:

    >>> l = MyList()
    >>> l
    []

    >>> l.append(0)
    >>> l
    [0]

    >>> l.append(5)
    >>> l.append(1)
    >>> l
    [0, 5, 1]

    >>> l.print_sorted()
    [0, 1, 5]

    >>> m = MyList()
    >>> m.append(4)
    >>> m.append(-3)
    >>> m.append(2)
    >>> m.append(1)
    >>> m
    [4, -3, 2, 1]

    >>> m.print_sorted()
    [-3, 1, 2, 4]
